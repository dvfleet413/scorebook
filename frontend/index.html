    About

    Scorebook is a Vanilla JS SPA with a RoR backend. This project was created as a challenge in creating a Javascript frontend to consume a Rails API, while using OOP with complex associations.

    The purpose of this app is to score a baseball game using a traditional scorecard. For simplicity, the app makes several assumptions about the baseball game being scored. Features may be added soon to allow for these cases:

    • There are no double plays or fielder's choices
    • In no half inning do more than 9 players have an at bat
    • There are no player substitutions

    All other situations are accounted for, including:

    • At Bat squares are updated based on form submission
    • Options to advance runners who are on base
    • Games automatically go into extra innings if tied after bottom of ninth, tenth, etc.
